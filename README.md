# ⚽ Football Turf Booking System - Frontend

This repository contains the **frontend application** for the **Football Turf Booking System**.

The frontend is developed using **React** and **Vite**, providing a responsive UI for browsing turf slots, booking football turfs, and user authentication.

---

## 🚀 Features

- User Registration & Login
- Browse Available Turf Slots
- Book Turf Slots Online
- Responsive UI
- Authentication-Based Booking

---

## 🛠️ Tech Stack

| Technology | Description |
|-----------|-------------|
| React | Frontend Framework |
| JavaScript | Programming Language |
| HTML & CSS | UI Structure & Styling |
